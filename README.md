# ROMIKEY - Digital-агентство полного цикла

Веб-страница для digital-агентства ROMIKEY, созданная по макету из Figma.

## Описание

Это современная, адаптивная веб-страница для digital-агентства с темной темой и синими акцентами. Страница включает в себя все необходимые разделы для представления компании, её услуг и команды.

## Особенности

- 🎨 **Современный дизайн** - Темная тема с синими акцентами
- 📱 **Полностью адаптивная** - Оптимизирована для всех устройств
- ⚡ **Интерактивность** - Анимации, hover-эффекты, плавная прокрутка
- 🎯 **SEO-оптимизированная** - Семантическая HTML разметка
- 🚀 **Быстрая загрузка** - Оптимизированные стили и скрипты

## Структура страницы

1. **Header** - Логотип, контакты, навигация
2. **О компании** - Описание и временная шкала развития
3. **Цель & Миссия** - Цели компании и ключевые метрики
4. **Почему мы лучше** - Преимущества агентства
5. **ROICONTEXT** - Профиль генерального директора
6. **Наша команда** - Специалисты компании
7. **Присоединяйся к нам** - Форма для соискателей
8. **Отзывы клиентов** - Отзывы и рекомендации
9. **Опыт специалистов** - Контактная форма
10. **Footer** - Контакты и дополнительная информация

## Установка и запуск

1. Скачайте все файлы проекта
2. Откройте `index.html` в браузере
3. Или запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

## Кастомизация

### Цвета
Основные цвета можно изменить в CSS переменных:
- Основной синий: `#0066ff`
- Темный фон: `#0a0a0a`
- Белый текст: `#ffffff`

### Шрифты
По умолчанию используется шрифт Inter. Для изменения замените ссылку в `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```

### Изображения
Замените placeholder'ы `.image-placeholder` на реальные изображения:
```html
<div class="image-placeholder">
    <img src="path/to/image.jpg" alt="Описание">
</div>
```
